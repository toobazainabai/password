# Password Strength Checker

A simple Python script that checks if a password is Weak, Moderate, or Strong based on these rules:

## Checks:

* Length must be **8+ characters**
* Must contain **at least one special character** (@, #, $, %, &, *, !)
* Must include **a number**
* Must include **an alphabet character**

## How to Run:

```
python password_checker.py
```

## Example Output:

Weak password
Moderate password
Strong password
